Siderodrome



Siderodrome was made for a prototyping and scripting class, in groups, students had to identify and recreate the main elements of a chosen game genre.
The prototype's goal was to isolate some key features of the thriller game genre and apply them.

Game Presentation

  • Title : Siderodrome
  • Studio : DESS NAD-UQAC
  • My role : Level designer & integrator
  • Genre : Horror game
  • Platforms : PC
  • Game modes : Thriller exploration

Tools Used

  • Game engine : Unreal Engine 5
  • Scripting langage : Unreal Engine blueprints
  • Documentation : Illustrator, Google Suite


My missions

  • Researching horror games staples and must-haves.
    In order to design a realistic horror game prototype I looked for different references in games, movies, show and books and compilled everything that made the experience scary, tense and horrific to create a strong base for the intentions, gameplay and atmosphere of the prototype.

  • Designing for a short prototype.
    This project being a short prototype for a class, I decided to focus on some precise points and mechanics that improves on the intention and main experience we want to focus on.

  • Creating the level.
    After choosing the gameplay elements and overall game design I went in the Unreal Engine to create the level and code some of the mechanics and elements myself.
